Full Stack

Sistema de Gestión Académica

Aplicación web institucional desarrollada con enfoque Full Stack para la gestión de procesos académicos universitarios. El proyecto integra frontend con Angular, consumo de APIs REST, pruebas de servicios con Postman, lógica backend con Laravel/PHP y manejo de datos mediante bases de datos relacionales.

Angular 21 TypeScript SCSS Laravel PHP Postman APIs REST MySQL

Funciones trabajadas

  • Diseño de vistas con hero institucional, cards, tablas y modales.
  • Componentes reutilizables para mantener consistencia visual.
  • Consumo de APIs REST desde Angular.
  • Pruebas y validación de endpoints mediante Postman.
  • Apoyo en lógica backend usando Laravel y PHP.
  • Manejo de información académica conectada a bases de datos.
  • Diseño responsive para escritorio, tablet y móvil.
