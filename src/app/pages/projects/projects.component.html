Proyecto principal

Proyecto académico-institucional desarrollado con Angular, enfocado en módulos universitarios, gestión académica y experiencia de usuario.

Angular 21

Sistema de Gestión Académica

Aplicación web institucional desarrollada con Angular para la gestión de procesos académicos universitarios. El proyecto incluye navegación por módulos, componentes reutilizables, búsqueda de estudiantes, vistas responsivas, consumo de servicios y diseño alineado a una identidad institucional.

Angular 21 TypeScript SCSS Angular Material APIs REST

Funciones trabajadas

  • Diseño de vistas con hero institucional y breadcrumbs.
  • Componentes globales como buscadores, cards, tablas y modales.
  • Integración de servicios para consultar información académica.
  • Manejo de rutas por módulos y roles.
  • Diseño responsive para escritorio, tablet y móvil.
